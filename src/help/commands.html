Any command transmitted to the server is put in a directory called 'Commands' in the root directory of the connected user. Fill in the command line field (main panel) by the requested command and press 'enter'. The server will send back a string like 'Command received: command_name" to confirm the command has been put in the correct directory.
Here is the first list of commands available for moving the diffractometer remotely.

  • Kill - stop abruptly the device
  • Restore - revert the analysis to the starting point
  • Status - show the device current status
  • Stop - stop the current measure
